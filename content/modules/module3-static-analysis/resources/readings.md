# Module 3: Readings and References

## Required Reading
- [Static Program Analysis](https://cs.au.dk/~amoeller/spa/) - Chapters 5-6: Dataflow Analysis (Free online textbook)

## Recommended Reading
- Aho, Lam, Sethi, Ullman. *Compilers: Principles, Techniques, and Tools* (Dragon Book) - Chapter 9: Machine-Independent Optimizations
- Nielson, Nielson, Hankin. *Principles of Program Analysis* - Chapters 1-2

## Key Papers
- Kildall, G. "A Unified Approach to Global Program Optimization" (1973) - The foundational paper on dataflow analysis frameworks
- Kam & Ullman. "Monotone Data Flow Analysis Frameworks" (1977) - Lattice-theoretic foundations

## Tool Documentation
- [SonarQube Rules](https://rules.sonarsource.com/) - See how industrial tools implement these analyses
- [Pylint Source Code](https://github.com/pylint-dev/pylint) - A real Python static analyzer

## Visualization
- [CFG Visualizer](https://dreampuf.github.io/GraphvizOnline/) - Use Graphviz DOT language to visualize your CFGs
