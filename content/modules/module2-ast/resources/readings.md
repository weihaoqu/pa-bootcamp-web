# Module 2: Readings and References

## Required Reading
- [Python `ast` Module Documentation](https://docs.python.org/3/library/ast.html)
- [Green Tree Snakes - the missing Python AST docs](https://greentreesnakes.readthedocs.io/)

## Recommended Reading
- [AST Explorer](https://astexplorer.net/) - Interactive AST visualization for many languages
- [Tree-sitter](https://tree-sitter.github.io/tree-sitter/) - Incremental parsing library used in modern editors
- Aho, Lam, Sethi, Ullman. *Compilers: Principles, Techniques, and Tools* (Dragon Book) - Chapter 5: Syntax-Directed Translation

## Tools to Explore
- **AST Explorer** (astexplorer.net) - Paste code, see its AST in real-time
- **Python `ast` module** - Built-in, used in all Module 2 exercises
- **Tree-sitter** - Used by Neovim, Helix, and other editors for syntax highlighting

## Design Patterns
- **Visitor Pattern:** Gamma et al., *Design Patterns* - Used extensively for AST processing
- **Interpreter Pattern:** Evaluating expression trees (our ExpressionEvaluator exercise)
